## ▶ How to Run

### Option 1: Run on Google Colab (Recommended)

Thus, the notebook can be executed directly in a browser without any local installation:

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)]
(https://colab.research.google.com/github/Dhawan4324/HTL-Free-CsSnCl-Perovskite-Solar-Cells/blob/main/Solar-Efficiency-ML.ipynb)

---

### Option 2: Run Locally (Jupyter / VS Code)

However, for users preferring a local environment:

1. Install Python (3.8+ recommended)
2. Install required packages:

   ```bash
   pip install -r requirements.txt
   ```
3. Open the notebook using:

   * Jupyter Notebook
   * JupyterLab
   * VS Code
