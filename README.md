# Cloud-Based Machine Learning Approach for Performance Prediction of HTL-Free CsSnCl₃ Perovskite Solar Cells

This repository contains the materials associated with the manuscript:
**"Cloud-Based Machine Learning Approach for Performance Prediction of HTL-Free CsSnCl₃ Perovskite Solar Cells"**.

## Contents
- `Solar-Efficiency-ML.ipynb` — Jupyter notebook for data processing, model training, and evaluation.

## How to run
1. Install Python (3.8+ recommended).
2. Install required packages (add `requirements.txt` if available).
3. Open the notebook:
   - Jupyter Notebook / JupyterLab / VS Code
